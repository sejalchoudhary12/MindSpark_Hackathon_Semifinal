# TrustAI – Financial Bodyguard

## Team Name
MindSpark

## Problem
Users receive scam calls, SMS, and fraud apps that steal money.

## Solution
TrustAI detects suspicious activity using AI and alerts users in real-time.

## Features
- Scam SMS detection
- Call risk detection (simulated)
- Trust Score dashboard
- Real-time alerts
- AI chat assistant
- Risk meter visualization

## Tech Stack
Frontend: React
Backend: Node.js
AI: NLP keyword detection
Database: Mock / JSON

## How to Run
1. Open deployed link
2. Test SMS scan
3. View risk score
